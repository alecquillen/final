Alec Quillen

Course Information

This course covers advanced topics on web and mobile applications with a focus on modern web application architectures and mobile-friendly user interfaces. Students will complete one major development project using the latest web technologies. CCSE IT Department.

Project Overview

  • Developed a movie search site using a movie API, featuring both grid and list views.
  • Implemented AJAX requests for dynamic content loading and user interactions.
  • Designed a responsive layout for optimal viewing on both desktop and mobile devices.
  • Enhanced user experience with modern UI elements and smooth transitions.
